Overview of Features

This assignment contains three canvases: The one with the green and red line is the initial canvas you draw on. After clicking generate SOR, the canvas on the right will show the original SOR. Then, you need to scroll down to the canvas on the bottom, which shows the output of assignment two. You can use the sliders to apply transformations to both shapes in the scene. Additionally, you can click the checkbox in order to switch between wireframe rendering and flat shading. I recommend you enable the endcaps for a better viewing experience when using the flat shading.

This assignment involved taking the output from the previous assignment, a .poly file and a .coor file, and doing two things to it: applying directional lighting and transformations to it. To do this, I first calculated all of the normals from the triangles. I did this by taking the first vertex, and calculating two vectors from the triangle. Then, I was able to use the cross product to calculate each normal of each triangle, which was then used to calculate the light of the surface after applying directional lighting. As for the transformations, I used the library provided by the textbook after asking Minghao if I could use their library for matrix operations. I used this library to calculate the transformation matricies for the given transformations, and then in the vertex shader I applied this transformation matrix to each vertex.
